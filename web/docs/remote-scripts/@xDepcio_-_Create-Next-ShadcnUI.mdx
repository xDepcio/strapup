### \@xDepcio/Create-Next-ShadcnUI
This script setps up minimalistic Next.js app with shadcn/ui. Steps:
    - Create Next.js app
    - Add shadcn/ui with Button component
    - Remove unwanted files generated by Next.js
    - Make initial commit

To run use:
```command
npx strapup run-script @xDepcio/Create-Next-ShadcnUI
```
### Executed commands
```bash
(projectName) => [
        `npx create-next-app@14.0.4 ${projectName} --ts --tailwind --app --no-eslint --no-src-dir --import-alias "@/*"`,
        `cd ${projectName}`,
        `npx strapup run-script @xDepcio/Add-ShadcnUI`,
        isUnix() ? `rm public/next.svg public/vercel.svg app/favicon.ico` : `rm public/next.svg public/vercel.svg, app/favicon.ico`,
        isUnix() ? `sed -i '/return (/,/)/c\    return (\\n        <main>\\n            <h1>Hello World!</h1>\\n        </main>\\n    )' app/page.tsx` : '',
        `git add .`,
        `git commit -m "Initial setup"`,
    ]
```
